Hexo搭建Github Pages

发表于 2020-02-23

之前按照github pages官方文档搭建了一个简易博客系统，发现没有可用的目录或者导航。偶然发现了一个博客框架，hexo，尝试了一下，还不错。

Quick Start

按照文档安装依赖，init了一个初始系统后。运行hexo server，就可以在本地开启博客系统
